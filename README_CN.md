## 👋简介
- 超级视窗是一款实验性应用程序，为大屏安卓设备提供更好的多任务体验。
- 支持横屏模式下的展开视图、本地应用启动器、可定制外观设置等功能。

## 🚀上手指南

1. [下载][downloads-url]并安装
2. 设置 Shizuku：
   - 安装并运行 Shizuku
   - 授予必要的权限
   - 将超级视窗连接到 Shizuku
3. 将设备旋转至横屏模式以启用展开视图
